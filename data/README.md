# TGF Datasets

## Parameters_5K.csv (Primary Dataset)

- **Source**: DCM (Dalmia Cement) cooling tower reports, India
- **Date range**: 2013-2025
- **Rows**: 5,614
- **Parameters**: 18

| Column | Unit | Description |
|--------|------|-------------|
| pH | - | Hydrogen ion concentration |
| Turbidity | NTU | Water clarity |
| Free_Residual_Chlorine | ppm | Biocide residual |
| TDS | ppm | Total Dissolved Solids |
| Total_Hardness | ppm as CaCO3 | Calcium + Magnesium hardness |
| Calcium_Hardness | ppm as CaCO3 | Calcium contribution |
| Magnesium_Hardness | ppm as CaCO3 | Magnesium contribution |
| Chlorides | ppm | Chloride ion concentration |
| Phosphate | ppm | Phosphate residual |
| Total_Alkalinity | ppm as CaCO3 | Bicarbonate + carbonate alkalinity |
| Sulphates | ppm | Sulphate ion concentration |
| Silica | ppm | Silica concentration |
| Iron | ppm | Iron (corrosion indicator) |
| Conductivity | uS/cm | Electrical conductivity |
| CoC | - | Cycles of Concentration |

## Parameters_1.csv

Smaller subset (1,000 rows) for quick testing.

## Data Provenance

Consolidated from 81 monthly cooling tower water analysis reports (Excel) from DCM/Dalmia Cement, SAC (Shriram Alkali & Chemicals), and related facilities. Reports cover Main Plant CWT, Power Plant CWT, and New Plant (X-01 CT) cooling towers.
